"FunkyPunkyAPI" 



La Funki Punky API es una API desarrollada para administrar la información de los personajes de la franquicia Funki Punky. Permite realizar operaciones de consulta, registro, edición y eliminación de personajes, además de gestionar el acceso mediante autenticación de usuarios.

El proyecto fue desarrollado utilizando Node.js, MySQL y JSON Web Tokens (JWT) para el manejo de la autenticación, ofreciendo una estructura organizada y una comunicación sencilla entre el backend y el frontend.

¿COMO EJECUTO EL PROYECTO?

Requisitos:

•Node Instalado

•Git Instalado (opcional)

•XAMPP y estar corriendo Apache + mySQL

Primeramente abrimos un CMD y nos ubicamos en "cd C:\xampp\htdocs"
Enseguida clonamos el repositorio con "git clone https://github.com/Lime0059/FunkyPunkyAPI".
En caso de utilizar Linux o no contar con Git, descarga el repositorio en formato .zip desde GitHub, extráelo y coloca la carpeta FunkyPunkyAPI dentro de /opt/lampp/htdocs/ si utilizas XAMPP para Linux. Después ejecuta el backend desde la carpeta backend con npm run dev


Enseguida, abrimos el proyecto en Visual Code y abrimos una terminal,. Nos ubicarmos en backend "cd backend". Despues correr el backend en el puerto con linea "npm run dev". En algunas maquinas no sirve el npm de node, simplemente se remplaza a "npm run dev"


El proyecto se ejecuta en "http://localhost/FunkyPunkyAPI" desde cualquier navegador. Deberias ser capaz de crear entradas y visualizar las entradas creadas.
