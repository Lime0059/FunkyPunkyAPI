"FunkyPunkyAPI" 



La Funki Punky API es una API desarrollada para administrar la información de los personajes de la franquicia Funki Punky. Permite realizar operaciones de consulta, registro, edición y eliminación de personajes, además de gestionar el acceso mediante autenticación de usuarios.

El proyecto fue desarrollado utilizando Node.js, MySQL y JSON Web Tokens (JWT) para el manejo de la autenticación, ofreciendo una estructura organizada y una comunicación sencilla entre el backend y el frontend.

¿COMO CORRO EL PROYECTO?
Abrir una terminal y ubicarse en backend "cd backend". Despues correr el backend en el puerto con linea "npm run dev"
EL proyecto se ejecuta en "http://localhost/FunkyPunkyAPI"
